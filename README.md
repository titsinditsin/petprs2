# Currency Converter Utility

Консольное приложение на языке C# для конвертации валют на основе фиксированных курсов.

### Технический стек
* **Runtime:** .NET 6.0+
* **Язык:** C#
* **Коллекции:** Dictionary<string, decimal> для хранения пар валют и их коэффициентов.

### Описание реализации
* **Валидация:** Использование метода `decimal.TryParse` для безопасного приведения типов и обработки некорректного пользовательского ввода.
* **Точность:** Применение типа данных `decimal` для исключения ошибок округления, характерных для финансовых вычислений.
* **Логика:** Итерация по словарю курсов для одновременного расчета стоимости в нескольких валютных парах (USD, EUR, CNY).
